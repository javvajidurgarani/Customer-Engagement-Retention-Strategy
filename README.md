 Customer Engagement & Product Utilization Analytics for Retention Strategy

 Overview

This project analyzes customer retention in the banking sector using customer engagement, product utilization, and relationship strength metrics.

Objectives

- Evaluate engagement vs churn
- Analyze product utilization impact
- Detect high-value disengaged customers
- Measure relationship strength
- Build churn prediction model

 Features

- Interactive Streamlit Dashboard
- Customer Segmentation
- KPI Monitoring
- Churn Analytics
- Machine Learning Prediction
- Feature Importance Analysis

 Tech Stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Scikit-Learn
- Matplotlib
- Seaborn

 Machine Learning

Random Forest Classifier

Accuracy: ~86%
